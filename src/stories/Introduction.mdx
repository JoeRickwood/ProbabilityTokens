# @garbo-succus/tokens

This is a collection of 3D assets for use with [Probability](https://probability.nz/).

## How to use these assets

To use an asset, reference it via "template":

```json
{
  "probability": "2024-01-01", 
  "states": [{
    "name": "Test game",
    "children": [
      {
        "template": "@garbo-succus/tokens/die",
        "position": [0, 0, 0]
      }
    ]
  }]
}
```
